Ebola Dataset for Mirador

Mirador dataset

This dataset is formatted as a project to load into Mirador. It is essentially a collection of CSV files, but with some extra information to make the navigation of the data easier and to display values such as dates and categories more conveniently.

The data includes 215 variables between demographic, clinical, laboratory (metabolic panel), and genetic. It was obtained by aggregation of the raw metadata as well as the SNP, iSNV and cluster data. The aggregation script is available here.
